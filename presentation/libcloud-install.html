Requirements

  1. Python 2.6 or 2.7
  2. Use virtual environments?
  3. Git?

Install via Git


$ git clone https://github.com/rackerlabs/multi-cloud-workshop.git
$ cd multi-cloud-workshop/libcloud/

Install via Zip

  1. Download multi-cloud-workshop-gh-pages.zip
  2. Unzip
  3. Change to multi-cloud-workshop-gh-pages/libcloud/

Install Dependencies


$ pip install -r requirements.txt
