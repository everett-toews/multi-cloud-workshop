jclouds Installation

  1. JDK 6+

jclouds Installation Using Git


$ git clone https://github.com/rackerlabs/multi-cloud-workshop.git
$ cd multi-cloud-workshop/jclouds/

jclouds Installation Using Zip

  1. Download multi-cloud-workshop-gh-pages.zip
  2. Unzip
  3. Change to multi-cloud-workshop-gh-pages/jclouds/

jclouds Installation


$ mvn dependency:copy-dependencies "-DoutputDirectory=./lib"
$ ls lib/
