fog Installation

  1. Ruby 1.9.3+
  2. RubyGems 2+
  3. Bundler 1.3.5+

fog Installation Using Git


$ git clone https://github.com/rackerlabs/multi-cloud-workshop.git
$ cd multi-cloud-workshop/fog/

fog Installation Using Zip

  1. Download multi-cloud-workshop-gh-pages.zip
  2. Unzip
  3. Change to multi-cloud-workshop-gh-pages/fog/

fog Installation


$ bundle install
